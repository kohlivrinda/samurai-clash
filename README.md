# samurai-clash
A simple 2D multiplayer fighting game in a retro pixel style created in 2022. 

Features to implement:
- high score
- fix sprites going offscreen


Controls:

Player 1:
- A : left
- D : right
- W : jump
- space : attack

Player 2:
- left : left
- right : right
- up : jump
- down : attack
